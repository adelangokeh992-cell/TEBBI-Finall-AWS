## 🏗️ الهيكل التقني المقترح

```
tebbi.com/                          → الصفحة الرئيسية
tebbi.com/booking                   → صفحة الحجز العامة (كل الأطباء)
tebbi.com/dr/{doctor-slug}          → بروفايل الطبيب + حجز
tebbi.com/clinic/{clinic-code}      → صفحة العيادة + أطباءها
tebbi.com/login                     → تسجيل الدخول
tebbi.com/super-admin               → لوحة مدير النظام
tebbi.com/clinic-admin              → لوحة مدير العيادة
tebbi.com/dashboard                 → لوحة الطبيب
```
