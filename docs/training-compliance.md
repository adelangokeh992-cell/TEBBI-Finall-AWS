# تدريب الفريق على الأمان والامتثال — TEBBI

للامتثال لـ HIPAA ومتطلبات التعامل مع البيانات الصحية:

## التدريب السنوي

- **الفئة المستهدفة:** المطورون، الإداريون، أي شخص يصل إلى بيانات المرضى أو بيئة الإنتاج.
- **المواضيع:**
  - التعريف بـ PHI ومتطلبات الخصوصية والأمان.
  - سياسات المشروع: عدم تخزين الأسرار في الكود، استخدام HTTPS، MFA إلزامي، الصلاحيات (RBAC)، سجلات التدقيق.
  - كيفية التعامل مع النسخ الاحتياطية والاستعادة.
  - سياسة الإبلاغ عن الاختراق (متى وكيف يُبلَّغ).
  - مراجعة خريطة بيانات PHI واتفاقيات BAA إن لزم.

## التوثيق

- توثيق مواعيد التدريب والمشاركين (الاسم، التاريخ، الموضوع) لأغراض المراجعة والامتثال.
- يمكن الاحتفاظ بسجل في ملف أو نظام إداري خارج المستودع (مثلاً جدول أو مستند داخلي).

## التردد

- يُنصح بإجراء التدريب **مرة سنوياً على الأقل** وعند انضمام أعضاء جدد للفريق الذين يصلون إلى بيانات حساسة أو بيئة الإنتاج.
